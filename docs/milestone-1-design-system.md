# Milestone 1 设计系统

状态：已按用户确认的 Android 主屏设计进入实现。

## 页面结构

- 四个主入口固定为：记录现场、待看、印记、设置；
- 使用同一个自定义底部 TabBar，不为不同页面替换图标或选中形态；
- 次级 Tab 按 Android 返回键时先回到“记录现场”，再次返回交给系统；
- 页面避让状态栏、底部手势区和 TabBar；
- Milestone 1 不实现 CRUD，尚未开放的操作以阶段提示反馈，不创建空跳转页。

## 色彩

| Token | 浅色 | 深色 |
|---|---|---|
| background | `#FFFFFF` | `#171311` |
| text | `#1B1715` | `#F8F2EE` |
| muted | `#6F6965` | `#B7AAA3` |
| accent | `#A74F17` | `#D6783F` |
| border | `#E9E4E0` | `#39312D` |

浅色模式背景保持真白，不改为米白。深色模式使用暖黑表面，不做简单反相。

## TabBar

- 图标：24dp、2dp 圆角线性描边；
- 顺序及隐喻：票券音符、书签、日历、齿轮；
- 所有状态保持同一图标轮廓；
- 选中态仅将图标和文字改为暖棕，并在顶部显示 32dp × 3dp 短横；
- 不使用圆形、胶囊、拨片底或实心替换图标作为选中效果。

## 迁移边界

设置页只保留 Android 本地备份与恢复的未来入口，不出现 iOS、iCloud、跨平台导入导出或手机间迁移入口。
